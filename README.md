# Start project OMS CMS

[Documentation](https://oms-cms.readthedocs.io/ru/latest/)
 
- Python => 3.6
- Django => 2
- Postgres == 10

**Установка:**

Создать виртуальное окружение ::

    python -m venv venv
    
Активировать виртуальное окружение

Установить CMS ::

    pip install oms-cms
    
Создать стартовый проект ::
    
    oms-start
