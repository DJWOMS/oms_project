# Start project OMS CMS
 
- Python => 3.6
- Django => 2
- Postgres == 10

**Установка:**

- Установить CMS ::

    pip install git+https://github.com/DJWOMS/oms_cms.git

- Создать стартовый проект. В "name" указать имя вашего проекта ::

    oms-start "name"

- Создание БД и настройка CMS ::

    python manage.py deployGFS
